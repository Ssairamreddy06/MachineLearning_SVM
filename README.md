# MachineLearning_SVM
A comparative study of SVM kernels (Linear, Polynomial, RBF) on nonlinear data using scikit-learn, with visualisations of decision boundaries and performance evaluation.
## 📘 Project Overview

This project explores how different kernel functions affect the performance of Support Vector Machines (SVMs) on nonlinear data. Using a synthetic dataset generated with scikit-learn, we compare Linear, Polynomial, and Radial Basis Function (RBF) kernels.

The project demonstrates how kernel choice influences decision boundaries, model flexibility, and classification accuracy. Visualisations and evaluation metrics such as accuracy scores, confusion matrices, and classification reports are used to analyse performance.

## 🎯 Objectives

- Understand the concept of Support Vector Machines (SVM)
- Explore the “kernel trick” and its importance
- Compare Linear, Polynomial, and RBF kernels
- Visualise decision boundaries for each model
- Evaluate model performance on training and test data

## 🛠️ Technologies Used

- Python
- NumPy
- Matplotlib
- scikit-learn

## 📊 Key Features

- Synthetic nonlinear dataset (make_moons)
- Data preprocessing using StandardScaler
- SVM implementation using pipelines
- Model comparison using accuracy metrics
- Decision boundary visualisation
